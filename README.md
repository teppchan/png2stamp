# PNG to Stamp

子供が描いた絵の PNG 画像から、スタンプ用の SVG / STL / OpenSCAD ファイルを作る小さなブラウザツールです。

## できること

- PNG を読み込む
- 2値化またはディザでスタンプ面に変換する
- 簡易ノイズ除去をする
- 線を少し太らせる、細らせる
- スタンプ面を SVG として保存する
- 持ち手付きの STL を直接保存する
- 持ち手付きの OpenSCAD ファイルを保存する

通常は `stamp_model.stl` をそのまま Creality Print や Cura に読み込めます。形状をあとで調整したい場合だけ OpenSCAD 版を使ってください。

## 使い方

1. `index.html` をブラウザで開きます。
2. PNG 画像を読み込みます。
3. 変換方式、しきい値、ノイズ除去、太らせる・細らせるを調整します。
4. スタンプサイズと持ち手サイズを調整します。
5. `STLを保存` を押します。
6. 保存した `stamp_model.stl` を Creality Print などに読み込みます。
7. 必要なら `SVGを保存` と `OpenSCADを保存` も使って、あとから形状調整します。

## SVG に変換する方針について

`PNG -> 前処理 -> SVG -> 3D` はかなり良い方針です。今回のツールはそこに加えて、同じ処理結果から直接 STL も作れるようにしています。線画は2値化、写真や濃淡のある画像はディザを選ぶと、黒い面積の密度でグレースケールを表現できます。理由は次の通りです。

- 2値化とノイズ除去を画像処理として独立して扱いやすい
- スタンプ面をあとで目視確認しやすい
- 3D 化の前に左右反転やサイズ調整をしやすい
- OpenSCAD などの 2D 押し出し系ツールと相性が良い
- まず STL を直接出してすぐ試作し、必要なときだけ SVG / OpenSCAD に戻れる

ただし、必ずしも「滑らかな輪郭トレース」が必要とは限りません。子供の絵は少しガタついた線の味があるので、まずはこのツールのように 2値化後の形を素直に使い、必要になったら Potrace などで輪郭トレースを追加するのがおすすめです。

## 注意点

- 文字をスタンプにしたい場合は左右反転をオンにします
- 写真で濃淡を残したい場合は変換方式を `ディザ` にします
- ディザの細かい点が消える場合はノイズ除去回数を 0 に下げます
- 線が細すぎると欠けやすいので、少し太らせる設定が有効です
- まずは 30mm から 40mm くらいの小さめサイズで試作すると調整しやすいです
